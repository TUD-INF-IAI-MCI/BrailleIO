IBrailleIOAdapterManager Methods

The IBrailleIOAdapterManager type exposes the following members.

Methods
  NameDescription
AddAdapter
Adds an adapter to the list of available adapters.
GetAdapters
Gets a list of all registered adapters.
RemoveAdapter
Removes an adapter form the list of available adapters.
Synchronize
Synchronizes the specified matrix with the Adapters. That means the given matrix will been send to the adapters. At least the active Adapter should display the matrix. Adapters that are able to set a "Synch" flag also should mirrow (display) the matrix.
