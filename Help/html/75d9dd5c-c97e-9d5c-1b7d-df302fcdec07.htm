IPannable Methods

The IPannable type exposes the following members.

Methods
  NameDescription
GetXOffset
Gets the offset in horizontal direction. Should normally be negative. Moves the position of the content by the offset. E.g. a negative offset should move the content to the left and the viewable region to the right.
GetYOffset
Gets the offset in vertical direction. Should normally be negative. Moves the position of the content by the offset. E.g. a negative offset should move the content to up and the viewable region down.
Move
Move the content in both direction simultaneous. So the vertical and horizontal offset will been set at the same time.
SetXOffset
Sets the offset in horizontal direction. Should normally be negative. Moves the position of the content by the offset. E.g. a negative offset should move the content to the left and the viewable region to the right.
SetYOffset
Sets the offset in vertical direction. Should normally be negative. Moves the position of the content by the offset. E.g. a negative offset should move the content to up and the viewable region down.
