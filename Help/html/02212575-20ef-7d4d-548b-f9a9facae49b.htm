RenderMatrix(IViewBoxModel, Boolean)
Puts the given content-matrix in a matrix that fits in the given view. The content-matrix placement is with aware of the given Box model and panning offsets. Borders are not rendered. If the content-matrix don't fit in the view, the overlapping content is ignored. If the content-matrix is smaller than the view, the rest is set to false. This renderer takes also care about the panning, which is set in the view if they is IPannable.
RenderMatrix(IViewBoxModel, Object)
Renders a content object into an boolean matrix; while true values indicating raised pins and false values indicating lowered pins
